## Linux Distributions

  • Arch Linux - A x86_64 linux distribution focused on simplicity, modernity, pragmatism, user centrality, and versatility.
  • Gentoo Linux - A highly flexible, source-based Linux distribution.
  • Manjaro Linux - A beginner friendly distribution based on Arch Linux.
  • KISS Linux - A x86_64 distribution with a focus on simplicity, sustainability and user freedom.

## Window Managers

  • dwm - A simple small, fast, highly customisable dynamic window manager for X.
  • Xmonad - A stable window manager for X written in the functional programming language Haskell.
  • i3wm - A tiling, stacking, and tabbing window manager for X written in C.

## Text Editors

  • Vim - A small, fast, and efficient keyboard-based text editor.
  • Code - OSS - Open Source repository for Visual Studio Code.
  • bedd - A super-small (78 KiB), fully functional terminal text editor with mouse support.

## Web Browsers

  • Firefox - Free and open-source web browser developped by Mozilla.
  • QuteBrowser - A free, open-source, cross-platform, keyboard-driven web browser.
  • w3m - A terminal text-based web browser.

## Games

  • Minecraft - A 3D open-world sandbox building game.
  • Sauerbraten - Old-School open source FPS with awesome retro-style graphics.

## Chess Stuff

  • Lichess - An ad-free, open-source chess server
  • Stockfish - Strong, fast, free, and open-source chess engine.
  • Peregrine - A not-so-strong, not-so-fast, free and open-source chess engine written by the one and only: me.
